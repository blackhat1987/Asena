Asena
==============

Description
--------------
Asena is a framework to create position independant shellcode. It can also used as a on-the-fly self upgrading software or build customized stubs with func-in-technology.

Done
--------------
- mutex-check
- installation routine
- startup routine
- creation of PE-file
- connection protocol
- client/server
